# Employee Manager JPA

A Spring Boot REST API for Employee Management.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Features
- Create Employee
- Get Employee
- Update Employee
- Delete Employee

## Run Locally

Clone the repository:

```bash
git clone https://github.com/muktikashyap301-arch/employee-manager-jpa.git
```

Configure MySQL in application.properties and run:

```bash
mvn spring-boot:run
```
